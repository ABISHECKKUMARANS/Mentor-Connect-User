# Mentor-Connect-User

## Description
Mentor-Connect-User is a user-focused application designed to connect mentors and mentees in a collaborative learning environment.

## Features
- User authentication and profile management
- Browse and search for mentors
- Request mentorship sessions
- View mentor availability and expertise
- Communication tools for mentor-mentee interaction
- Session scheduling and tracking
