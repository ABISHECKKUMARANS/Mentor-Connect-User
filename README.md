# Mentor-Connect-User

## Description
Mentor-Connect-User is a user-focused application designed to connect mentors and mentees in a collaborative learning environment.

## Features
- User authentication and profile management
- Browse and search for mentors
- Request mentorship sessions
- View mentor availability and expertise
- Communication tools for mentor-mentee interaction
- Session scheduling and tracking

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager

### Installation
1. Clone the repository
```bash
git clone https://github.com/ABISHECKKUMARANS/Mentor-Connect-User.git
cd Mentor-Connect-User
```

2. Install dependencies
```bash
npm install
```

3. Set up environment variables
Create a `.env` file in the root directory and configure necessary variables

4. Start the application
```bash
npm start
```
